HTML Cheatsheet

Tag Name Description
<head> Head Defines the head, the metadata of the webpage, including links and tab title.
<body> Body Defines the body, the visible structure of the webpage.
<footer> Footer Defines the footer, the bottom of the webpage, contact details and other links may be placed here.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> Headings Displays headings in HTML, range from <h1>-<h6>, with <h1> the largest and <h6> the smallest.
<p> Paragraph Used to write paragraphs on a webpage.
Tag Name Description
<header> Header Defines the header, the top of the webpage, a navigation bar may be placed here
<section> Section Defines a section, this creates standalone sections of webpage, which include topics and themes that relate with eachother.
<article> Article Defines contents of a webpage that is independent from the general flow of the webpage, such as a quote.
<footer> Footer Defines the footer, the bottom of the webpage, contact details and other links may be placed here.
Tag Name Description
<strong> Bold Wraps around text to make it bold in the HTML.
<em> Italic Emphasises text and puts it in italics
<span> Span A tag that goes inline and marks up part of the text. You can then apply CSS to that specific part, such as a different colour compared to the rest of the text.
<br> Line Break Creates a line break in text.
